# Pre-Submission Checklist

## Paper Quality
- [ ] All placeholders replaced (author name, email, mentor)
- [ ] All figure labels match \ref calls
- [ ] Figures embedded (PNG for LaTeX, inline for HTML)
- [ ] PDF compiles without errors
- [ ] Bibliography complete and formatted
- [ ] Acknowledgments section complete
- [ ] Data availability statement included
- [ ] Ethics statement reviewed

## Reproducibility
- [ ] All results reproducible at $0 cost
- [ ] Requirement files pinned
- [ ] Docker build passes
- [ ] CI passes on GitHub
- [ ] Tests pass (11/11)

## OpenRouter Data
- [ ] `results.json` downloaded from Colab/Kaggle
- [ ] `python3 paper/auto_update_paper.py` run
- [ ] Tables populated with real values
- [ ] Figures regenerated with real data
- [ ] Final PDF compiled

## Submission
- [ ] arXiv metadata prepared (title, authors, cs.CL category)
- [ ] arXiv endorsement secured (or alternative)
- [ ] License chosen (CC-BY 4.0)
- [ ] Repository made public
- [ ] DOI minted via Zenodo

## Final Review
- [ ] Spell-check complete
- [ ] All sections flow logically
- [ ] Contributions clearly stated
- [ ] Limitations honestly described
- [ ] Figures legible in print
