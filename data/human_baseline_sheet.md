## After Collection: Analysis Script

Once you have ratings from 5 people, run this:

```python
import json, statistics

# Enter the 50 scores from each rater here:
raters = {
    "Rater1": [...50 scores...],
    "Rater2": [...50 scores...],
    "Rater3": [...50 scores...],
    "Rater4": [...50 scores...],
    "Rater5": [...50 scores...],
}

# Compute mean per item
items = 50
item_means = []
for i in range(items):
    scores = [raters[r][i] for r in raters]
    item_means.append(statistics.mean(scores))

print(f"Human mean: {statistics.mean(item_means):.3f}")
print(f"Human std: {statistics.stdev(item_means):.3f}")

# Save
with open("results_rootcause/human_baseline.json", "w") as f:
    json.dump({"raters": raters, "item_means": item_means}, f, indent=2)
print("Saved to results_rootcause/human_baseline.json")
```
