Key Findings

📐 Bias is Learned

Scoring bias emerges during instruction tuning — 2.09× amplification in rubric order bias. We can fix it by changing how we fine-tune.

🔗 Biases Compound

Position + verbosity biases produce 1.5-2.1× worse effects than additive predictions. Single-bias tests underestimate real-world impact.
