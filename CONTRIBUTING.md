# Contributing

## Development

```bash
git clone https://github.com/ssamba1/Scoring-Bias-in-LLM-as-a-Judge
cd research-draft
pip install -r requirements.txt
make test
```

## Standards

- **Python**: 3.11+, type hints encouraged
- **Style**: flake8 + black (run `make lint`)
- **Tests**: Add tests for new analysis scripts
- **Paper**: Edit `paper/camera_ready_full.tex`, then compile
- **Commit messages**: Describe the change, not the files changed
