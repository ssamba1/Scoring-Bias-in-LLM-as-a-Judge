Where Does Scoring Bias Come From?

A base vs instruct comparison revealing that LLM instruction tuning improves format robustness but increases content sensitivity — the first causal evidence that scoring bias is modulated by instruction tuning.

Key Findings

−44%

Rubric Order Bias Decreases

Instruction-tuned models are 44% less susceptible to rubric ordering effects than base models. Format processing improves.

−77%

Score ID Bias Decreases

Score label manipulation (numeric vs letter) affects instruct models 77% less. Label parsing improves.

+35%

Reference Answer Bias Increases

Content-based bias worsens by 35% after instruction tuning. Exemplar sensitivity increases — a previously unknown effect.

Consistent

Pattern Holds Across 3 Model Families

Llama 3 8B (−33%), Mistral 7B (−25%), and Gemma 2 2B (−55%) all show the same differential pattern.
