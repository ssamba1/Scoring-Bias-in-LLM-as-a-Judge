## BibTeX

```bibtex

  title     = {Scoring Bias in {LLM-as-a-Judge} Models:
               A 22-Model Landscape with Base-Instruct Comparison},
  author    = {Samba, Sricharan},
  journal   = {arXiv preprint},
  year      = {2026},
  doi       = {10.5281/zenodo.21361920},
```

## Citation File Format (CFF)

The `CITATION.cff` file in the repository root provides machine-readable citation metadata:

```yaml
cff-version: 1.2.0
title: "Scoring Bias in LLM-as-a-Judge Models: A 22-Model Landscape with Base-Instruct Comparison"
authors:
  - family-names: Samba
    given-names: Sricharan
doi: 10.5281/zenodo.21361920
repository-code: "https://github.com/ssamba1/Scoring-Bias-in-LLM-as-a-Judge"
license: MIT
message: "If you use this work, please cite it."
```

GitHub will automatically render this file with a "Cite this repository" button on the repository page.

## Exporting from Reference Managers

Most reference managers support importing via DOI directly:

1. Go to https://doi.org/10.5281/zenodo.21361920
2. Use your reference manager's "Import by DOI" feature
3. The metadata will be downloaded automatically

### Zotero

1. Click the "Add Item by Identifier" button (magic wand icon)
2. Enter: `10.5281/zenodo.21361920`
3. Zotero fetches all metadata automatically

### Mendeley

1. Click "Add" → "Add Entry Manually" or use the Web Importer
2. Enter the DOI
3. Details populate automatically

### EndNote

1. Go to References → New Reference
2. Enter the DOI in the DOI field
3. Use "Update References" to fetch metadata
