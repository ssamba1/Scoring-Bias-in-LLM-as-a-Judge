Key Findings

📉 −44% Format Bias (Rubric Order)

Instruction tuning reduces sensitivity to rubric scale direction. Instruct models parse scoring instructions more robustly than base models.

📉 −77% Format Bias (Score ID)

The strongest format improvement. Score label format (numeric, letter, descriptive) matters far less for instruct models.

📈 +35% Content Bias (Reference Answer)

Content bias increases with instruction tuning. Instruct models are more susceptible to exemplar-based biasing.

🔬 IIAR Hypothesis

Instruction-Induced Attention Redistribution explains the differential effect: more attention to all prompt features helps format parsing but hurts content robustness. 5 testable predictions.
