Key Finding

−44%

Rubric Order Bias Decreases

Instruction-tuned models are 44% less susceptible to rubric ordering. Format processing improves.

−77%

Score ID Bias Decreases

Score label manipulation affects instruct models 77% less. Label parsing improves.

+35%

Reference Answer Bias Increases

Content-based bias worsens by 35%. Exemplar sensitivity increases — previously unknown.

3/3

Pattern Holds Across ALL Families

The differential effect is consistent across 44 model families — from 410M to 13B parameters.
